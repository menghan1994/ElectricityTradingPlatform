# Epic 5: 日前报价决策工作台

**目标：** 交易员可在"一屏决策"工作台中审核AI报价建议、逐时段微调并记录理由、确认最终报价方案；可批量查看多电站申报状态并快速切换；所有覆盖操作完整记录。

**FRs覆盖：** FR7, FR8, FR10, FR50, FR51

**相关NFRs：** NFR5(10并发)

**UX需求：** 三栏布局（电站列表+96时段网格+Agent面板）、多电站标签页切换、Period96Grid键盘导航、快捷键支持

---

## Story 5.1: 多电站状态概览与申报跟踪

As a 交易员（李娜）,
I want 在电站列表视图中批量查看所负责电站的关键状态并按状态筛选,
So that 我能快速掌握今天哪些电站需要处理，优先处理未完成的。

**Acceptance Criteria:**

**Given** 交易员已登录
**When** 进入交易工作台页面
**Then** 左侧展示该交易员所负责电站的列表，每个电站显示：电站名称、今日申报状态、最新预测偏差、储能SOC（配有储能的电站）

**Given** 电站列表展示中
**When** 交易员输入电站名称关键字
**Then** 列表按名称筛选

**Given** 电站列表展示中
**When** 交易员按申报状态筛选（未开始/AI建议已生成/交易员已审核/已提交）
**Then** 列表仅显示符合所选状态的电站

**Given** 每日申报状态
**When** 报价流程推进
**Then** 状态按顺序更新：未开始→AI建议已生成→交易员已审核→已提交，状态变化实时反映在列表中

## Story 5.2: 96时段报价微调与覆盖记录

As a 交易员（李娜）,
I want 对AI报价建议进行逐时段微调并记录调整理由,
So that 我能根据自己的经验判断优化报价方案，且每次调整都有据可查。

**Acceptance Criteria:**

**Given** 交易员选择一个电站且AI建议已生成
**When** 交易员在96时段报价网格（Period96Grid）中点击某时段的报量或报价单元格
**Then** 进入编辑模式，交易员可修改数值

**Given** 交易员修改了某时段的报量或报价
**When** 交易员保存修改
**Then** 系统弹出调整理由输入框（必填），交易员填写后保存成功

**Given** 交易员已修改某时段
**When** 修改保存成功
**Then** 系统记录覆盖操作（调整内容、调整前值、调整后值、理由、时间、操作人），被调整的时段用橙色背景+编辑图标+"调"Tag标识

**Given** Period96Grid
**When** 交易员使用键盘操作
**Then** 支持方向键在单元格间移动、Enter进入/保存编辑、Esc取消编辑、Tab跳到同行下一可编辑列

## Story 5.3: 报价方案确认与提交

As a 交易员（李娜）,
I want 完成所有时段审核后确认最终报价方案,
So that 报价方案被正式提交，触发后续储能调度流程。

**Acceptance Criteria:**

**Given** 交易员已审核完所有96个时段
**When** 交易员点击"确认提交"（或Ctrl+Enter快捷键）
**Then** 系统将申报状态更新为"已提交"，自动记录确认时间戳和操作人

**Given** 交易员点击确认提交
**When** 存在时段数据缺失（报量或报价为空）
**Then** 系统阻止确认并提示具体缺失时段编号

**Given** 报价方案确认成功
**When** 该电站配有储能设备
**Then** 系统自动触发储能调度指令生成流程

**Given** 报价方案确认成功
**When** 确认操作完成
**Then** 确认操作（时间戳+操作人+操作内容）写入审计日志

---

## Story 5.4: 多电站快速切换与状态保持

As a 交易员（李娜）,
I want 在多个电站之间快速切换，每个电站保持各自独立的报价状态,
So that 我能高效处理多个电站的报价工作而不丢失操作进度。

**Acceptance Criteria:**

**Given** 交易员正在处理电站A的报价
**When** 交易员点击电站列表中的电站B（或使用Alt+左/右箭头快捷键切换）
**Then** 工作台切换到电站B的报价面板，电站A的操作状态（已修改的时段、未保存的编辑）保持不变

**Given** 交易员切换回电站A
**When** 查看电站A的报价面板
**Then** 之前的操作状态完整保留（已微调时段标识、Agent面板展开状态等）

**Given** 交易员管理5个以上电站
**When** 使用Alt+1~5快捷键
**Then** 可快速切换到第1-5个电站

**Given** 多电站切换
**When** 前端状态管理
**Then** Pinia Store按电站ID索引保持各站独立状态，切换仅改变activeStationId
